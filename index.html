Contact Manager

This is a single page application (SPA) that saves contacts you enter to local storage. The libraries used for this application are knockout.js, jquery and bootstrap for the responsive layout.

To add a contact, click the "Add a Contact" button. To edit an existing contact, change the values in the input fields in the table and click the "Save Contacts" button. To remove a contact, clck the "remove" link next to each contact. To remove all contacts and delete local storage click the red "Clear All" button.
